In the example that we started with, the function is \(f(x)=x+7\). We can sum up the process that allows us to count the sequence by saying ``there is a one-to-one correspondence between the lists \begin{equation*} (7, 8, 9, \ldots , 44 ) \end{equation*} and \begin{equation*} (0, 1, 2, \ldots , 37 ) \end{equation*} and the later has \(38\) entries.''
