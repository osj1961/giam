To re-iterate, we've shown that the power set of \(\Naturals\) is equivalent to the interval \((0,1)\) which is one of the sets whose cardinality is c. So the continuum hypothesis, the thing that got Georg Cantor so very heated up, comes down to asserting that \begin{equation*} \aleph_1 = \mathbf{ c}. \end{equation*}
