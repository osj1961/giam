The domain of a function (or binary relation) is the set of numbers appearing in the first coordinate. The range of a function (or binary relation) is the set of numbers appearing in the second coordinate. Consider the set \(\{0,1,2,3,4,5,6\}\) and the function \(f(x) = x^2 \pmod{7}\). Express this function as a relation by explicitly writing out the set of ordered pairs it contains. What is the range of this function? \hint{ \begin{equation*} f \; = \; \{(0,0), (1,1), (2,4), (3,2), (4,2), (5,4), (6,1)\} \end{equation*} \begin{equation*} \Rng{f} \;= \; \{0,1,2,4\} \end{equation*} }
