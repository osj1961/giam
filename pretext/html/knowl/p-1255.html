If \(x_1\) and \(x_2\) have opposite parity, we will assume w.l.o.g. that \(x_1\) is even and \(x_2\) is odd. The equality \(f(x_1)=f(x_2)\) becomes \(x_1/2 = -(x_2-1)/2\). Note that \(x_1 \geq 2\) so \(f(x_1) = x_1/2 \geq 1\). Also, note that \(x_2 \geq 1\) so \begin{gather*} x_2 - 1 \geq 0\\ (x_2-1)/2 \geq 0\\ -(x_2-1)/2 \leq 0\\ f(x_2) \leq 0 \end{gather*} therefore we have a contradiction since it is impossible for the two values \(f(x_1)\) and \(f(x_2)\) to be equal while \(f(x_1) \geq 1\) and \(f(x_2) \leq 0\).
