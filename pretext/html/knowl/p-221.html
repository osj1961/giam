An immediate consequence of this last equation is that \(a^2\) is an even number. Using the lemma above we now know that \(a\) is an even integer and hence that there is an integer \(m\) such that \(a=2m\). Substituting this last expression into the previous equation gives \begin{equation*} (2m)^2 = 2 b^2, \end{equation*} thus, \begin{equation*} 4m^2 = 2 b^2, \end{equation*} so \begin{equation*} 2m^2 = b^2. \end{equation*}
