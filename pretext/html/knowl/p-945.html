Okay, so on to an example. In SectionÂ 4.1 we discovered a formula relating the sizes of a set \(A\) and its power set \({\mathcal P}(A)\). If \(|A| = n\) then \(|{\mathcal P}(A)| = 2^n\). What we've got here is an infinite family of logical sentences, one for each value of \(n\) in the natural numbers, \begin{equation*} |A| = 0 \implies |{\mathcal P}(A)| = 2^0, \end{equation*} \begin{equation*} |A| = 1 \implies |{\mathcal P}(A)| = 2^1, \end{equation*} \begin{equation*} |A| = 2 \implies |{\mathcal P}(A)| = 2^2, \end{equation*} \begin{equation*} |A| = 3 \implies |{\mathcal P}(A)| = 2^3, \end{equation*} et cetera.
