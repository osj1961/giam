Suppose that \(p\) is a prime, for each \(n\) in \(\Integers^+\), define the set \(P_n = \{ x \in \Integers^+ \suchthat \, p^n \divides x \}\). Conjecture and prove a statement about the containments between these sets. \hint{When \(p=2\) we have seen these sets. \(P_1\) is the even numbers, \(P_2\) is the doubly-even numbers, etc.}
