Proof

We proceed by induction on \(n\).

Basis: When \(n = 1\) the sum has only one term, \(1^2 = 1\). On the other hand, the formula is \(\displaystyle \frac{1(1+1)(2\cdot 1+1)}{6} = 1\). Since these are equal, the basis is proved.

Inductive step:

\begin{minipage}{4 in} Before proceeding with the inductive step, in this box, we will figure out what the right-hand side of our theorem looks like when \(n\) is replaced with \(k+1\): \begin{gather*} \frac{(k+1)((k+1)+1)(2(k+1)+1)}{6}\\ = \frac{(k+1)(k+2)(2k+3)}{6}\\ = \frac{(k^2+3k+2)(2k+3)}{6}\\ = \frac{2k^3+9k^2+13k+6}{6}. \end{gather*}

By the inductive hypothesis, \begin{equation*} \sum_{j=1}^k j^2 = \frac{k(k+1)(2k+1)}{6}. \end{equation*}

Adding \((k+1)^2\) to both sides of this equation gives \begin{equation*} (k+1)^2 + \sum_{j=1}^k j^2 = \frac{k(k+1)(2k+1)}{6} + (k+1)^2. \end{equation*}

Thus, \begin{equation*} \sum_{j=1}^{k+1} j^2 = \frac{k(k+1)(2k+1)}{6} + \frac{6(k+1)^2}{6}. \end{equation*}

Therefore, \begin{gather*} \sum_{j=1}^{k+1} j^2 = \frac{(k^2+k)(2k+1)}{6} + \frac{6(k^2+2k+1)}{6}\\ = \frac{(2k^3+3k^2+k)+(6k^2+12k+6)}{6}\\ = \frac{2k^3+9k^2+13k+6}{6}\\ = \frac{(k^2+3k+2)(2k+3)}{6}\\ = \frac{(k+1)(k+2)(2k+3)}{6}\\ = \frac{(k+1)((k+1)+1)(2(k+1)+1)}{6}. \end{gather*}

This proves the inductive step, so the result is true.
