Given that we can't give a precise definition of a real number at this point it is perhaps surprising that we can define the set \(\Complexes\) of complex numbers with precision (modulo the fact that we define them in terms of \(\Reals\)). \begin{equation*} \Complexes = \{ a + bi \suchthat a \in \Reals \; \mbox{and} \; b \in \Reals \; \mbox{and} \; i^2 = -1 \} \end{equation*}
