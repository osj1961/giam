To describe the rational numbers precisely we'll have to wait until Section 1.6. In the interim, we can use an intuitively appealing, but somewhat imprecise definition for the set of rationals. A rational number is a fraction built out of integers. This also provides us with a chance to give an example of using the main other way of describing the contents of a set — so-called set-builder notation. \begin{equation*} \Rationals = \{ \frac{a}{b} \suchthat a \in \Integers \; \mbox{and} \; b \in \Integers \; \mbox{and} \; b \neq 0 \} \end{equation*}
