Proof

(By mathematical induction)

Basis: Clearly \(3 \divides 6\).

Inductive step:

(We need to show that \(3 \divides (k^3 + 2k + 6) \; \implies \; 3 \divides ((k+1)^3 + 2(k+1) + 6\).)

Consider the quantity \((k+1)^3 + 2(k+1) + 6\). \begin{gather*} (k+1)^3 + 2(k+1) + 6\\ = (k^3 + 3k^2 + 3k + 1) + (2k + 2) + 6\\ = (k^3 + 2k + 6) + 3k^2 + 3k + 3\\ = (k^3 + 2k + 6) + 3(k^2 + k + 1). \end{gather*}

By the inductive hypothesis, 3 is a divisor of \(k^3 + 2k + 6\) so there is an integer \(m\) such that \(k^3 + 2k + 6 = 3m\). Thus, \begin{gather*} (k+1)^3 + 2(k+1) + 6\\ = 3m + 3(k^2 + k + 1)\\ = 3(m + k^2 + k + 1). \end{gather*}

This equation shows that 3 is a divisor of \((k+1)^3 + 2(k+1) + 6\), which is the desired conclusion.
