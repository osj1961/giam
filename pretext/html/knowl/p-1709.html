Hopefully you used the quadratic formula to solve the previous exercise. You should of course have found two solutions, \(-1.618033989\ldots\) and \(.618033989\ldots\), these decimal approximations are actually \(-\phi\) and \(1/\phi\), where \(\displaystyle \phi = \frac{1+\sqrt{5}}{2}\) is the famous “golden ratio”. If we are hoping for the sum over all the occupied positions of \(r^k\) to be convergent, we need \(|r|\lt 1\), so the negative solution is extraneous and so the inequality \(r^{k+2} + r^{k+1} \; \leq \; r^k\) is true in the interval \([1/\phi, 1)\).
